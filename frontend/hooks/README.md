# Hooks

Ce dossier contiendra les custom hooks React réutilisables.

## Hooks à créer :

- `use-local-storage.ts` : Hook pour persistence localStorage
- `use-debounce.ts` : Hook pour debouncing (recherche)
- `use-scroll-position.ts` : Hook pour position de scroll
