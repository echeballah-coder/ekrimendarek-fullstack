# Types

Ce dossier contiendra toutes les interfaces et types TypeScript globaux.

## Fichiers à créer :
- `index.ts` : Export central
- `entities.ts` : Entités principales (User, Vehicle, Booking, Agency, Review)
- `api.ts` : Types pour les requêtes/réponses API
- `forms.ts` : Types pour les formulaires
- `enums.ts` : Enums (VehicleType, BookingStatus, etc.)
