# EkriMenDarek - Backend

Ce dossier contiendra l'application backend construite avec **NestJS**.

## Stack Technique Prévue

- **Framework** : NestJS
- **Langage** : TypeScript
- **Base de données** : PostgreSQL
- **ORM** : Prisma
- **Authentification** : JWT

## Structure Future

- `src/` : Code source
- `test/` : Tests E2E et unitaires
- `prisma/` : Schéma de base de données
